```sh

```

Serveur local

```sh
npm install

```


### Features

-  Build basic Angular app with webpack
-  Simple twitter application
-  Sass support
-  Typescript support
-  ES6 modules support
-  Webpack support
-  Optimized build package
-  Minimal and straightforward setup
-  Watches code and refreshes browser with latest changes automatically
-  Sourcemap support
